# notebooks

Notebooks that contain step-by-step analysis, modeling, and results.

## Files

- `gender_detection.ipynb` — Predicts gender from text features using classical NLP pipelines.
- `submission.csv` — Submission-ready predictions generated by the notebook.

## How to Use

Open the notebooks and run cells in order. Ensure the `data` folder is present and matches expected filenames.
This module produces a `submission.csv` file suitable for evaluation or leaderboard submission.
